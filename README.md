# 🧟 Resident Evil: Requiem – Themed Web Experience

A cinematic, interactive **Resident Evil-inspired website** built using modern frontend tools and animations.

This project was created as a **learning experiment with GSAP**, combining horror aesthetics, smooth animations, and creative UI design to deliver an immersive web experience.

---

## 🌐 Live Demo

🚀 **Experience it here:**
👉 https://resident-evil-requiem-sp1k.vercel.app/

---

## 🎯 About The Project

Inspired by *Resident Evil: Requiem*, this project focuses on recreating the **dark, atmospheric feel** of the franchise through web technologies.

Rather than copying an existing design, this project blends:

* 🎨 Personal creativity
* 🎬 Cinematic animations
* 🧠 Experimentation with GSAP

---

## ✨ Features

* 🎞️ Smooth GSAP animations
* 🧟 Horror-themed UI/UX design
* 🎥 Video background integration
* ⚡ Interactive elements
* 📱 Responsive layout (optimized for different screens)
* 🎮 Game-inspired immersive sections

---

## 🛠️ Tech Stack

* **React.js**
* **GSAP (GreenSock Animation Platform)**
* **Tailwind CSS**
* **JavaScript (ES6+)**
* **Vite**

---

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/RohanGhosh69/Resident-Evil-Requiem.git
```

### 2️⃣ Navigate into the project

```bash
cd Resident-Evil-Requiem
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the development server

```bash
npm run dev
```
